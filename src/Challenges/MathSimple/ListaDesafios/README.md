:green_circle: Nível 1 — Fundamentos

1. Par ou Ímpar
Leia um número inteiro e informe se é par ou ímpar.

2. Maior de Dois
Leia dois números e mostre o maior.

3. Média Simples
Leia 3 notas e calcule a média.

4. Contagem Crescente
Mostre os números de 1 até N (N informado pelo usuário).

5. Soma de 1 até N
Some todos os números de 1 até N.

:yellow_circle: Nível 2 — Condicionais e Laços

6. Fatorial
Calcule o fatorial de um número N.

7. Tabuada
Mostre a tabuada de um número de 1 a 10.

8. Contar Vogais
Receba uma string e conte quantas vogais existem.

9. Número Primo
Verifique se um número é primo.

10. Inverter Número
Receba um número inteiro e inverta seus dígitos.
Ex: 1234 → 4321

:orange_circle: Nível 3 — Arrays e Lógica

11. Maior do Array
Dado um array de inteiros, encontre o maior valor.

12. Soma de Pares
Some apenas os números pares de um array.

13. Remover Duplicados
Dado um array, retorne outro sem valores repetidos.

14. Segundo Maior
Encontre o segundo maior número do array.

15. Rotacionar Array
Rotacione um array para a direita em K posições.
Ex: [1,2,3,4], k=1 → [4,1,2,3]

:red_circle: Nível 4 — Intermediário Real

16. Palíndromo
Verifique se uma string é palíndromo (ignore espaços e maiúsculas).
Ex: "Ame a ema" → true

17. Frequência de Elementos
Conte quantas vezes cada número aparece em um array.

18. Dois que Somam Alvo (Two Sum)
Dado um array e um alvo, retorne os índices de dois números que somam o alvo.

19. Compressão de String
Comprima a string contando repetições consecutivas.
Ex: "aaabbc" → "a3b2c1"

20. Validador de Parênteses :fire:
Verifique se uma string com ()[]{} é válida.
Ex válidos: "()[]{}", "([{}])"
Ex inválidos: "(]", "([)]"
